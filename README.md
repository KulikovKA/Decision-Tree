# Decision-Tree
Описаны следующие классы : **Node** ,**Decision Tree** ,**GradientBoosting** , **RandomForest**


### 1. Decision Tree 
Реализован полный цикл построения дерева регрессии:
*   **Критерий разбиения:** Variance Reduction (снижение дисперсии / MSE).
*   **Рекурсивное построение:** Метод `build_tree`.
*   **Гиперпараметры:** `max_depth`, `min_samples_split`, `min_samples_leaf`, `max_features`.
*   **Визуализация:** Кастомный метод `plot_tree_horizontal_tree` для отрисовки структуры дерева через `matplotlib`.

### 2. Gradient Boosting 
*   Последовательное обучение деревьев на анти-градиенте функции потерь (в данном случае — на остатках `y - y_pred`).
*   Использует "слабые" деревья (shallow trees) в качестве базовых эстиматоров.
*   Реализован `learning_rate` для контроля вклада каждого дерева.

### 3. Random Forest 
*   Реализован **Bootstrap Aggregating**: обучение на случайных подвыборках с возвращением.
*   Усреднение предсказаний множества независимых деревьев.
